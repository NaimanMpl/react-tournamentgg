## Getting Started

### Installation
1. Clone the repo
```sh
git clone https://github.com/NaimanMpl/react-tournamentgg.git
```
2. Install npm packages
```sh
cd server
npm install

cd client
npm install
```
3. Launch server and client
```sh
cd server
npm run dev
cd client
npm run dev
```
